{/* The movability constraint is

[a b]
[c d]

(a > 0 and d > 0) or (b > 0 and c > 0)

So the question is: what is the likelihood that this statement is true, across all choices of rows and columns in the matrix?

We can represent the expected mass in each cell as

= row_margin * col_margin / total_mass

Notice that this means the expected value of each row and col are equal to their margins, which we like.

The way we can represent the PMF of each cell in the matrix is with a Hypergeometric Distribution. (Link to Wikipedia). For our case, we actually only care about the case where x=0, because all of our entries are non-negative, so our P(x=0) simplifies.

Num successes (k) = 0 (we will be taking 1 - our result)
Population size (N) = T (total mass)
Number of draws (n) = row value
Number of successes in population (K) = col value

= \frac{{ K \choose k } { N - K \choose n - k }}{{ N \choose n }}
= \frac{{ T - col \choose row }}{{ T \choose row }}
= \frac{(T - col)!(T - row)!}{T! (T - col - row)!}

One thing to notice here is that the resulting P(x=0) is symmetric. It doesn't matter if we swap the columns and rows. This should be the case, if our math is correct, because permuting the matrix shouldn't change its inherent sparsity.

To calculate the sparsity of the mixing for a given set of constraints, we can then calculate this P value for each cell, and then iterate over all of the possible 2x2 subs, averaging

P(can move) = 1 - (1 - a * d) * (1 - b * c) = bc + ad - abcd

---

To use this value, we would divide the base number of mixes (NxM) by the sparsity (1 - E[can move]). Then, we ensure that even with very sparse matrices, we still do an appropriate number of mixing steps to get approximately uniform outputs.
